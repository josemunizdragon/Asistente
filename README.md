# PetAssistant

Demo de asistente virtual: backend ASP.NET Core Web API + frontend Expo/React Native.

- **Backend:** `backend/` — API con auth fake, perfil, asistente, health.
- **Frontend:** `mobile/pet-assistant-mobile/` — Login, Signup, Home, Avatar (Expo).

## Versión

**v1.0.0** — Commit inicial estable.

## Subir a GitHub

1. Crea un repositorio nuevo en GitHub: https://github.com/new  
   - Nombre: `PetAssistant` (o el que quieras)  
   - Sin README, sin .gitignore (ya están en el proyecto)

2. Asigna la URL a `origin` y haz push (sustituye `TU_USUARIO` por tu usuario de GitHub):

```bash
cd /Users/jose/Documents/VisualStudio/PetAssistant
git remote set-url origin https://github.com/TU_USUARIO/PetAssistant.git
git push -u origin main
git push origin v1.0.0
```
